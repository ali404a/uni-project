# 🌌 درب التبانة الجامعي

منصة عراقية تساعد طلبة السادس الإعدادي على اختيار التخصص والجامعة المناسبة بعد إعلان النتائج، مبنية على بيانات وزارة التعليم العالي الرسمية (الحدود الدنيا للقبول المركزي 2024/2025 والكليات الأهلية 2025/2026).

مصمّمة لتتحمّل **10,000+ طالب** عبر قاعدة بيانات Supabase.

---

## المزايا

- **المحاكاة الذكية**: يُدخل الطالب معدله فيرى أقرب 40 قسماً و10 معاهد لمعدله، مع طباعة/تحميل استمارة A4.
- **الأقسام**: مجمّعة حسب التخصص المعياري (الطب البشري، هندسة الحاسوب…) — الضغط على تخصص يعرض كل الجامعات ومعدلاتها، والضغط على جامعة يعرض تفاصيلها.
- **الجامعات الأهلية**: بأقسامها ومعدلاتها وأجورها السنوية.
- **لوحة تحكم كاملة** (`/admin.html`): إحصائيات الزوار والاستخدام، تعديل الحدود الدنيا، إضافة/تعديل/حذف الأقسام والجامعات، ترتيب الجامعات الأهلية بالسحب، تعديل الأجور، وإدارة الأخبار.

## التشغيل محلياً (بلا قاعدة بيانات)

```bash
npm install
npm start
```
يفتح على http://localhost:3000 — يعمل ببيانات `data/seed.json` في الذاكرة.
لوحة التحكم: http://localhost:3000/admin.html (المستخدم: `admin` / كلمة المرور: `darb2026`)

---

## الربط بـ Supabase (للإنتاج — 10,000+ طالب)

### 1) أنشئ مشروع Supabase
اذهب إلى [supabase.com](https://supabase.com) → New Project (مجاني).

### 2) أنشئ الجداول
في Supabase → **SQL Editor** → الصق محتوى `server/db/schema.sql` → Run.

### 3) اضبط المفاتيح
انسخ `.env.example` إلى `.env` واملأ:
```
SUPABASE_URL=https://xxxxx.supabase.co       # من Settings → API
SUPABASE_ANON_KEY=eyJ...                       # anon public key
SUPABASE_SERVICE_KEY=eyJ...                     # service_role key (سرّي!)
JWT_SECRET=نص-عشوائي-طويل-غيّره
```
(المفاتيح في Supabase → Settings → API)

### 4) ارفع البيانات
```bash
npm run seed
```
يرفع كل الجامعات والأقسام والمعدلات والمعاهد، ويُنشئ مستخدم الأدمن.

### 5) شغّل
```bash
npm start
```

## الرفع على Vercel

1. ارفع المشروع على GitHub.
2. في Vercel → Import Project → اختر المستودع.
3. أضف متغيرات البيئة (`SUPABASE_URL`, `SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_KEY`, `JWT_SECRET`) في Settings → Environment Variables.
4. Deploy.

الإعداد موجود في `vercel.json`.

## لوحة التحكم — كيف تعدّل كل شيء

| ما تريد تعديله | من أين |
|---|---|
| الحد الأدنى للقبول لأي قسم | الأقسام والمعدلات → أيقونة التعديل |
| إضافة/حذف قسم | الأقسام والمعدلات |
| إضافة/تعديل/حذف جامعة | الجامعات |
| ترتيب الجامعات الأهلية | الأهلية والأجور → اسحب |
| أجور الأقسام الأهلية | الأهلية والأجور → اختر جامعة → عدّل |
| الأخبار | الأخبار |
| مشاهدة الزوار والاستخدام | الإحصائيات |

> **مهم**: غيّر كلمة مرور الأدمن بعد الربط عبر تعديل `ADMIN_PASSWORD` في `.env` قبل تشغيل `npm run seed`، أو من جدول `admin_users` في Supabase.

---

## الإحصائيات — خفيفة بلا تأثير على السرعة

- التتبّع **fire-and-forget**: الطلب يُرسَل بلا انتظار الرد، فلا يؤخّر الصفحة.
- تُخزّن **عدّادات يومية مُجمّعة** (صف واحد لكل يوم) بدل تسجيل كل زيارة منفردة.
- الزوّار الفريدون عبر **بصمة مجهّلة** (تجزئة SHA-256) — بلا تخزين أي بيانات شخصية أو IP خام.
